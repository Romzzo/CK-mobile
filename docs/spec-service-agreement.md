# 이용약관 페이지 구현 명세

## 라우트
`src/app/service/agreement/page.tsx`

## UX 방향 (토스 스타일 개선)

아코디언 구조에 얽매이지 말고 아래 원칙 우선. 구조는 자유롭게 선택.

### 1. 상단 TL;DR 요약 카드
약관 전문 전에 핵심 3~5줄 먼저 노출. 배경색 카드(`bg-surface rounded-2xl p-4`).

> 예시 문구:
> - 이용기간 종료 후 다운로드한 콘텐츠는 즉시 삭제 필수
> - 이용 승낙일로부터 7일 이내 청약철회 가능 (단, 다운로드 시 불가)
> - 무단복제·라이선스 초과 사용 시 2배 손해배상
> - 콘텐츠 저작권은 회사 귀속 — 사용 권한만 부여

### 2. 중요 조항 callout 박스
`<b>` 또는 `<u>` 태그로 강조된 조항은 별도 callout 스타일 적용.
배경: `bg-amber-50` 또는 `bg-brand/5`, 좌측 보더: `border-l-2 border-brand`, `rounded-lg p-3 my-2`

### 3. 목차 앵커 내비게이션
아코디언 대신 상단에 챕터 목차 pill 버튼 → 탭하면 해당 섹션으로 smooth scroll.
```
[총칙]  [회원가입]  [서비스 이용]  [손해배상]
```
스타일: `flex gap-2 overflow-x-auto`, pill: `rounded-full border border-line px-3 py-1 text-[12px] shrink-0`

### 4. 발효일 + 버전 표기
푸터에 발효일 명시. 스타일은 `text-[11px] text-ink-mute`.

### 5. 타이포 위계
- 조항 제목(`<strong>`): `text-[14px] font-bold text-ink block mt-5 mb-1`
- 본문(`<li>`): `text-[13px] text-ink-soft leading-relaxed`
- 강조(`<b>` 태그 inside `<li>`): callout 박스로 분리 또는 `text-ink font-medium`

## AccordionSection 컴포넌트

- 헤더: `제N장 챕터명` + 우측 chevron (▼/▲)
- 기본 상태: **모두 열림** (open)
- 클릭 시 토글 (열기/닫기)
- 애니메이션: 부드러운 expand/collapse
- 헤더 스타일: `text-[15px] font-bold text-ink` + `border-b border-line py-4`
- 본문 스타일: `text-[13px] text-ink-soft leading-relaxed`

### 본문 렌더링

내용이 방대(~16,000자)하므로 `dangerouslySetInnerHTML`로 HTML 문자열 주입.
콘텐츠는 `src/data/agreementContent.ts`에 챕터별 HTML 문자열로 분리.

본문 HTML 요소 스타일 (`prose` 클래스 또는 직접 지정):
- `strong`: `font-semibold text-ink text-[13px]` + 블록으로 표시 (`display: block; margin-top: 1rem; margin-bottom: 0.25rem`)
- `strong.u > u` (중요 조항 강조): 동일하게
- `ol`, `ul`: `list-decimal` / `list-disc`, `pl-4 space-y-1`
- `li > ol`: 중첩 리스트, `pl-3`
- `b`: `font-semibold`

### 제2장 회원가입 및 이용계약

```html
<strong>제4조 (회원 이용계약의 성립)</strong>
<ol>
  <li>회원의 이용계약은 이용자가 본 약관 및 "라이선스 규정", 개인정보처리방침에 동의하면서 제5조에 따른 이용신청을 하고, 회사가 이용을 승낙함으로써 성립합니다. 회원의 이용계약의 성립시기는 회사의 승낙이 회원에게 도달한 시점으로 하되, 유료회원의 경우 해당 서비스에 따른 회원 가입비의 입금확인이 되면 이용계약이 성립합니다.</li>
  <li>회원의 본 약관에 대한 동의는 이용신청 시 회사 사이트의 회원가입 절차 중 약관 동의 메뉴에서 "이용약관에 동의합니다."를 체크함으로써 의사표시를 합니다.</li>
  <li>민법상 미성년자가 유료 서비스를 이용할 경우, 결제 전 법정대리인의 동의를 받아야 하며, 그렇지 않았을 경우 본인 또는 법정대리인이 계약을 취소할 수 있습니다.</li>
</ol>
<strong>제5조 (회원의 이용신청)</strong>
<ol>
  <li>회원으로 가입하고자 하는 이용자는 반드시 회사에서 요청하는 제반정보(성명 또는 상호, 연락처, 전자우편, 사업자등록번호 등)를 사실대로 정확하게 등록하여야 합니다. 회사에 등록한 제반정보와 일치하지 않을 경우 이용자는 일체의 권리를 주장할 수 없습니다.</li>
  <li>타인의 명의나 ID, 비밀번호를 도용하여 이용신청을 한 회원의 모든 ID는 삭제되며, 관계법령에 따라 처벌을 받을 수 있습니다.</li>
  <li>회사는 회사에서 제공하는 서비스를 이용하는 회원에 대하여 콘텐츠의 사용 목적과 용도 등에 따라 등급별로 구분하여 가입비, 서비스 관련 사항에 차등을 둘 수 있습니다.</li>
  <li>본사(본점, 본원) 이외의 지사(지점, 분원, 영업소 등)를 별도로 두고 있다면, 지사(지점, 분원, 영업소 등)는 본사와 별도로 구분하여 가입을 해야 합니다.</li>
  <li>타사 납품/대행을 하는 경우에는 납품하는 최종 광고주의 업체 규모를 기준으로 라이선스를 취득하여야 합니다.</li>
</ol>
<strong>제6조 (회원 ID부여 및 서비스 이용기간)</strong>
<ol>
  <li><b>회사는 회원에게 한 개의 ID만을 제공합니다. 개인(개인사업자 포함)의 경우 1인 또는 1개의 사업자등록번호 기준으로 1개의 ID를, 법인 또는 단체의 경우 1개의 사업자등록번호를 기준으로 1개의 ID를 제공합니다.</b></li>
  <li>법인 또는 단체의 경우 유료회원 등급에 따라 5개 멀티로그인(동시접속)이 가능하며, 더 많은 동시접속을 원하는 회원은 추가 회원가입 또는 추가 라이선스 구매를 하여야 합니다.</li>
  <li>회원ID는 원칙적으로 변경이 불가하며, 부득이한 사유로 인하여 변경하고자 하는 경우에는 해당ID를 해지하고 재가입하여야 합니다.</li>
  <li>무료회원의 이용기간은 탈퇴 시까지이며, 유료회원의 이용기간은 가입한 회원의 구분에 따라 다르며 "라이선스 규정"에 명시되어 있습니다.</li>
  <li>라이선스 기간이 정상적으로 개시되어 이용을 시작한 이후, 본 약관에서 달리 정하는 경우를 제외하고는 임의로 이용기간을 정지할 수 없으며, 콘텐츠를 다운로드 받은 이후에는 하위 등급으로 라이선스 타입 및 기간을 변경할 수 없습니다.</li>
  <li>모든 유료회원은 이용기간이 만료된 이후에는 무료회원으로 자동 전환됩니다. 또한 이용기간 동안 회사 사이트에서 다운로드 받은 일체의 콘텐츠(폰트 프로그램 포함)는 모두 즉시 삭제하여야 하며, 어떠한 형태로도 이용기간 이후 사용할 수 없습니다.</li>
  <li>다만 이용기간 동안 제작한 제작물은 이용기간이 만료된 후에도 재사용 또는 편집·변형, 재업로드 되지 않는 범위 내에서 그대로 게시해 둘 수 있습니다.</li>
</ol>
<strong>제7조 (회원 탈퇴 및 자격 상실 등)</strong>
<ol>
  <li>회원이 이용기간 중 탈퇴를 원할 경우 사이트 관리자에게 요청할 수 있으며, 회사는 확인 절차를 거쳐 즉시 회원탈퇴를 처리합니다.</li>
  <li><b>회원이 다음 각호의 사유에 해당하는 경우 및 제12조 제8항의 회원의 의무 위반에 해당하는 경우 회사는 즉시 회원자격을 제한 및 정지시킬 수 있습니다. 이 때, 회원은 가입비의 반환을 청구할 수 없으며, 회사가 손해를 입은 경우 그 손해까지 배상해야 합니다.</b>
    <ol>
      <li>가. 가입 신청 시에 허위 내용을 등록한 경우</li>
      <li>나. 서비스 이용과 관련하여 회원이 부담하는 채무를 기일에 지급하지 않는 경우</li>
      <li>다. 제12조 제8항의 제가호 내지 제아호에 해당하는 경우</li>
    </ol>
```
